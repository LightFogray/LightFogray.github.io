第一题

为什么 [‘1’, ‘7’, ‘11’].map(parseInt) 返回 [1, NaN, 3]

答案解析：

要点: 函数参数
  • map()函数

    map是Array原型中的一个方法，它返回一个新的数组，其结果是将原始数组的每个元素传递给一个函数。
    当一个函数传递到map()，对于每次迭代，都有三个参数传递到函数：currentValue(元素值)，currentIndex(元素下标)，array(原数组)。
    即当[‘1’, ‘7’, ‘11’]数组中的第一个元素’1‘到map执行parseInt时，parseInt接收的参数为('1', 0, ['1', '7', '11'])
    所以可知，
    parseInt依次接收参数为：

  • parseInt()函数

    parseInt函数有两个参数：string和radix（进制）。如果提供的radix为空或者为假值，进制（基数）默认为10。
    所以

    parseInt只接受前两个参数

    所以最后返回为[1,NaN,3]
